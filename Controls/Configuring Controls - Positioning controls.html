Configuring controls : Positioning

Each control can be configured in certain ways.

Each control has an options object that we can set in map options object. For example, the mapTypeControl has mapTypeControlOptions, the zoomControl has the zoomControlOptions etc.

The control options can have a style and a position.

Here we shall see how to position our controls. The position property of the map control options object will let this happen.

The position property has a value of a ControlPosition. The ControlPosition object has a set of 12 values to specify position information on the map.
