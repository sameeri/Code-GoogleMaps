Exploring the various style options for controls

The controls available natively can be styled. They have specific style options available and each one of them is an enumeration constants stored in objects. They have a naming convention [ControlName]ControlStyle. So for example, the scale control has it's style options in google.maps.ScaleControlStyle.

We would love to explore the options. Here, we are going to see all options output to the browser, console windows. No map display. Only JavaScript code and object love. We can observe the options are actual properties with a number associated with each one of them.
