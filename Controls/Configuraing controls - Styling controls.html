Configuring controls : Styling controls

Each control can be configured in certain ways.

Each control has an options object that we can set in map options object. For example, the mapTypeControl has mapTypeControlOptions, the zoomControl has the zoomControlOptions etc.

The control options can have a style and a position.

Styles are predifined for the native controls. For example the zoom control has it's styles predefined in the google.maps.ZoomControlStyle object of the API.

For example we can change the mapType control to be a dropdown menu for selecting the map type. We set the style property of the mapControlOptions object. We set style to google.maps.MapTypeControlStyle to DROPDOWN_MENU.

Instead of a slider, we can have +/- for zoom options. We can change the zoom control to have +/- by setting the style property of the zoomControlOptions object. We set style to google.maps.ZoomControlStyle.SMALL.
