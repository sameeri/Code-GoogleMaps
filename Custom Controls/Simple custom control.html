Adding our own controls.

map.controls is an array of google.maps.ControlPosition objects. Each ControlPosition stores an MVCArray of the controls displayed in that position. The API places controls at each position by the order of an index property; controls with a lower index are placed first. For example, two custom controls at position TOP_CENTER will be laid out according to this index order, with lower index values taking precedence.

We can access the MVCArray of controls at a particular position, by passing the position as the index for the controls array. Example: map.controls[google.maps.ControlPosition.TOP_CENTER] will give us a reference to the MVCArray of controls..We can add our control to this array using the push command.
