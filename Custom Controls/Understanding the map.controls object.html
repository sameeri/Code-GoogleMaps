Understanding the map.controls object!

The map object has a controls property. This is an array of ControlPosition objects.

Since controls can be placed at different locations on the map, each ControlPosition corresponds to the location that controls can be placed at.

Each position can have 0 or more controls placed there. So each control position has an array of controls at the position. This is an MVCArray according to Google.

We would like to observe these arrays, in this code sample. Understanding this, we can try to add our own custom control to this array. Total Awesomeness!

There are a total of 14 allowed and available positions. So the controls array should have 14 elements in it's array.
