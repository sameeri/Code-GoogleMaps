Let's go ahead and create a simple map.

We are going to create one for Norman, Oklahoma.

We first create a Map options object, and have a container for the map to display in. We call the constructor google.maps.Map, and pass in the html container element and the map options as arguments.

The div container hosting the map should have a height and width. I have set a 800x800 but you can have the whole page display the map. Just make sure, the elements html, body and the map container are all styled with 100% height.
