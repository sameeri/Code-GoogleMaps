What do we need to create a Google Map?

We need a map options object. It's just a plain object with certain properties. There are two required properties. center and zoom.

center takes a google.maps.LatLng object as it's value. We can new up an instance of the google.maos.LatLng constructor by passing the Latitude and Longitude..

zoom takes a number as it's value. zoom values go from 0 on. Higher zoom values zoom into the map.
